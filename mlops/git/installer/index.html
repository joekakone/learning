Téléchargement#

Bash est disponible en téléchargement gratuit, vous pouvez le télécharger ici https://www.bash.org/downloads/.

Installation#

Parcourez votre explorateur pour trouver le setup récemment téléchargé. Exécutez-le puis suivez les instructions pour finaliser l'installation.

Démarrage de Git Bash#

Une fois l'installation terminée, recherchez puis lancer Git Bash.
