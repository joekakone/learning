Interaction Python & SQLite3#

SQLite est le système de gestion de bases de données relationnelles orienté fichier. Nous verrons connecter se connecter à une base de donnée SQLite et effectuer les opérations CRUD.

Prerequisites#

No requirements

Packages#

# !pip install -q pandas

import sqlite3
import pandas as pd

Paths & Config#

DATABASE = "algojungle.db"

Connection to database#

conn = sqlite3.connect(database=DATABASE)

# Create a cursor
cur = conn.cursor()

Interaction with database#

0. Create a table#

sql1 = "DROP TABLE IF EXISTS customers;";

sql2 = """
CREATE TABLE customers
(
    DepartmentID INTEGER PRIMARY KEY NOT NULL,
    Name VARCHAR(100),
    GroupName VARCHAR(100)
);
"""

cur.execute(sql1)
cur.execute(sql2)

conn.commit()
