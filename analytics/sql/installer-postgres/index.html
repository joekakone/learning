Installer Postgres et pgAdmin#

Postgres est le système de gestion de bases de données Open Source le plus populaire.

Installer Postgres#

1. Télécharger le setup#

Pour télécharger le setup, rendez-vous sur le site de EB Entreprise qui distribue les fichiers officielles de Postgres https://www.enterprisedb.com/downloads/postgres-postgresql-downloads

2. Installer Postgres#

Parcourez votre explorateur pour trouver le setup récemment téléchargé. Exécutez-le puis suivez les instructions pour finaliser l'installation.

Installer pgAdmin#

https://www.pgadmin.org/download/pgadmin-4-windows/

Lancer pgAdmin#

Une fois l'installation terminée, recherchez puis lancer pgAdmin.

pgAdmin se lance

Connexion à Postgres#

Copiez et collez ce code dans l'éditeur SQL 

CREATE DATABASE algojungle;

CREATE TABLE customers
(
    customer_id INTEGER NOT NULL PRIMARY KEY,
    cutomer_name VARCHAR(35),
    cutomer_address VARCHAR(100),
    cutomer_birthday DATE
);
