La première version de Power BI a été publiée en 2015. Power BI a rapidement évolué pour devenir aujourd'hui l'un des outils d'analyse et de visualisation de données les plus populaires au monde. Power BI est propriétaire et est développé par Microsoft qui effectue des mises à jour régulières à raison d'une fois par mois pour Power BI Desktop. Il n'est disponible que sur Windows 10+.

Power Platform#

Power BI fait partie de la Power Platform de Microsoft qui est une suite logicielle low-code destinée à accélérer la productivité en entreprise.

  • Power BI
  • Power Apps
  • Power Pages (récement ajouté)
  • Power Automate
  • Power Virtual Agents

Ces outils permettent aux entreprises d'automatiser les tâches récurrentes et de se concentrer sur le core-business. Si vous souhaitez en savoirs plus, merci de vous rendre sur le site officel.

Power BI#

Power BI n'est pas un logiciel, il s'agit d'un ensemble. Power BI démocratise l'analyse de données qui pouvait nécessiter des compétences de pointes réservées aux spécialistes. L'outil a connu un grand succès ces dernières années et est adopté par un grand nombre d'entreprises.

Les composants#

Power BI est un ensenble de solutions.

  • Power BI Desktop
  • Power BI Service
  • Power BI Mobile
  • Power BI Report Builder

Power BI Desktop#

Power BI Desktop est l'outil de conception de rapports. Il est gratuit, disponible en téléchargement sur le site de Microsoft https://powerbi.microsoft.com/en-us/downloads/ et uniquement disponible pour les utilisateurs de Windows.

Power BI Service#

Power BI Service est une plateforme cloud (en ligne) qui permet de partager des rapports Power BI et de collaborer. Une fois les rapports conçus dans Power BI Desktop, il faut le publier sur le Power BI Service afin qu'il soit accessible aux utilisateurs. Power BI Service permet également de concevoir des rapports comme dans Power BI Desktop.

Power BI Mobile#

Power BI Mobile est une client qui permet d'accéder aux rapports et Dashboard Power BI en lecture uniquement. Il est disponible pour Smartphone et Tablette.

Power BI Report Builder#

Report Builder est l'outil de concetion des raports paginés Power BI. Pour votre information, les rapports paginés répondent bien aux besoin d'impression de tableaux qui s'étallent sur plusieurs pages.

Le flux de travail#

Le flux de travail avec Power BI peut être décliné en quatre étapes principales à savoir

  1. Préparation des données
  2. Modélisation des données
  3. Conception du rapports
  4. Partage du rapport

C'est une nouvelle façon de penser l'architecture des données.
