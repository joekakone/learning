Lire un fichier - Écrire dans un fichier#

Lire le contenu d'un fichier#

>>> f = open('filename.txt', 'r')
>>> content = f.read()
>>> content

>>> f.close()

Écrire dans un fichier#

>>> f = open('filename.txt', 'w')
>>> content = 'Python est le langage le plus cool !'
>>> f.write(content)
>>> f.close()
