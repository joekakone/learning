Les débuts du langage#

La première version du langage Python a été publiée en 1991 par Guido van Rossum (20 février 1991). Ah oui, un seul homme a créé le langage Python 😳.

Démocratiser la programmation informatique était l'un des principaux objectifs de Guido qui a conçu Python de telle sorte qu'il soit un langage facile à prendre en main 🤣. D'aucuns vous diront que Python est facile et c'est vrai.

Un langage interprété et haut niveau#

Python est un langage interprété et de type haut niveau. Sa syntaxe est très intuitive et facile à apprendre. Les messages d'erreur de Python sont très cordiaux, vous allez adorer 😁.

En tant que langage haut niveau, certaines fonctionnalités complexes sont implémentées nativement, ce qui représente un gain de temps considérable comparativement à un langage bas niveau comme le C, d'où la facilité qui est régulièrement évoquée.

Python est open-source et gratuit, vous pouvez le télécharger ici https://www.python.org/downloads/.

Python aujourd'hui#

Trente ans après sa publication, Python s'est imposé comme l'un des langages les plus utilisés et le plus utilisé dans certains domaines. En effet Python jouit d'une grande communauté qui contribue à son développement avec plusieurs projets.

Domaines d'application#

Voici quelques domaines dans lesquels Python est utilisé:

  • Logiciels bureau
  • Logiciels CLI/système
  • Développement web
  • Calcul scientifique
  • Science des données
  • Intelligence artificielle
  • Robotique
  • Les systèmes embarqués

Cette liste n'est pas exhaustive, Python est partout.

Python Software Foundation#

Le développement du langage Python est aujourd'hui piloté par la Python Software Foundation (PSF) dont le président est Guido van Rossum.

Vous trouverez beaucoup de détails sur la page Wikipédia de Python.
