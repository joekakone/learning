Installer Anaconda#

Je vous recommande d'utiliser TensorFlow avec Anaconda. Si vous n'avez pas encore Anaconda installé, suiviez ce tutoirel pour l'installer Installer Anaconda.

Créer un environnement virtuel#

Afin de préserver votre environnement de base, je vous recommande fortement de créer un environnement virtuel. Lancez Anaconda Prompt et faites la commande suivante: 

conda create --name tf
Cette commande va créer une environnement virtuel avec le nom tf.

Activer l'environnement virtuel#

Une fois l'environnement virtuel créé, il faut l'activer avec la commande suivante. 

conda activate tf

Installer TensorFlow#

Tensorflow peut être installé depuis le répertoire Conda ou PyPi. L'installation avec conda est préférable. 

conda install tensorflow
En cas d'éventuelles erreurs, vous pouvez essayer l'installer avec pip. 
pip install tensorflow

Vérifier la version#

Lancez Python et exécutez les instructions suivantes: 

import tensorflow as tf
from tensorflow import keras
print(tf.__version__)
print(keras.__version__)

Le code source de TensorFlow est disponible sur GitHub tensorflow/tensorflow.
